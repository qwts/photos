Lucide icon glyph for all Overlook iconography — never hand-draw SVGs; use this.

```jsx
<Icon name="lock" size={16} />
<Icon name="cloud-upload" color="var(--accent-amber)" />
```

Requires `<script src="https://unpkg.com/lucide@0.462.0/dist/umd/lucide.min.js"></script>` on the page. Standard sizes: 14 (inline/meta), 16 (buttons), 20 (toolbar).
