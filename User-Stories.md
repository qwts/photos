# User Stories

Planning home for the photos milestone / user-story breakdown. Canonical
user-story planning pages live here as feature work begins.

## Status

No product milestones are planned yet — the
[project-foundation epic (#1)](https://github.com/qwts/photos/issues/1) comes
first. When feature planning starts:

- Break work into **milestones** (`M01`, `M02`, …), each with its own wiki page
  (`User-Story-M01-<Title>`), and index them in the table below.
- Individual stories are filed as GitHub issues with the
  [User Story template](https://github.com/qwts/photos/issues/new?template=user-story.md)
  (scope / out-of-scope / exit criteria / acceptance scenarios /
  planning-discipline checklist / coverage review / open questions) and the
  `user story` label.
- A story's acceptance scenarios feed the testing lanes per the
  [Testing Strategy](Testing-Strategy) — coverage travels with the change.
